# Twitter Tweet UI

This project is a simple user interface (UI) representation of a Twitter tweet. The UI is designed to visually replicate the look and feel of a typical tweet on Twitter.

## Features

- User avatar
- Username and handle
- Tweet content
- Timestamp
- Like, retweet, and reply buttons
